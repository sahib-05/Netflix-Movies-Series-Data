# Netflix Movie Data Analysis

##  Overview

This project focuses on analyzing Netflix movie data to understand content trends, movie distribution, ratings, release patterns, and genre popularity using Python-based data analysis techniques.

The objective of this project is to perform data cleaning, explore the dataset, and generate meaningful business insights through visualizations and statistical analysis.

---

##  Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* CSV Dataset

---

##  Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values and duplicate records
* Analysis of movie release years
* Rating distribution analysis
* Genre/category analysis
* Country-wise content distribution
* Duration and runtime analysis
* Most frequent directors and cast analysis
* Visualization of Netflix content trends

---

##  Dataset Features

The dataset includes:

* Show ID
* Title
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre (Listed In)
* Type (Movie / TV Show)
* Description

---

##  Key Insights

### Content Type Distribution

Netflix contains both Movies and TV Shows, with Movies forming the major portion of the platform content.

### Release Year Trends

Most Netflix content was released after 2010, showing a strong increase in modern content production and platform expansion.

### Ratings Analysis

TV-MA and TV-14 are among the most common content ratings, indicating a strong focus on mature and teen audiences.

### Genre Popularity

Drama, International Movies, and Comedies are among the most frequently occurring genres on Netflix.

### Country-wise Distribution

The United States contributes the highest number of titles, followed by countries like India and the United Kingdom.

### Duration Analysis

Most movies fall within the standard runtime range, while TV Shows are commonly categorized by seasons.
